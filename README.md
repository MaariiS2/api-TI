# 📝 Task Manager API

Uma API simples para gerenciamento de tarefas, permitindo criar, listar, atualizar e excluir tarefas de forma prática.

---

## 🚀 Como executar

Siga os passos abaixo para rodar o projeto localmente:

```bash
# Clone o repositório
git clone https://github.com/MaariiS2/api-TI.git

# Acesse a pasta do projeto
cd api-TI

# Instale as dependências
npm install

# Execute o projeto
npm run dev
